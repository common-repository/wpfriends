Simple plugin to keep track of your friends posts

== Description ==

Simple plugin to keep track of your friends posts, 
similar to the friends page of LiveJournal.
Currently you can just add WordPress and LiveJournal feeds.

WORK IN PROGRESS!

== Installation ==

1. Upload this folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
